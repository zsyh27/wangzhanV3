# 内容审核Checklist

## 内容质量审核

### 1. 字数要求
- [ ] 文章字数在1500-3000字之间
- [ ] 内容充实，无冗余信息
- [ ] 内容完整，涵盖所有要点

### 2. 关键词密度
- [ ] 核心关键词密度在2%-3%之间
- [ ] 关键词自然融入，无堆砌
- [ ] 长尾关键词合理分布

### 3. 结构完整性
- [ ] 包含标题（H1）
- [ ] 包含6个标准部分（根据文章类型）
  - 产品选型类：问题背景、技术参数、选型建议、安装注意事项、常见问题、联系我们
  - 安装指导类：安装前准备、安装步骤、安装注意事项、常见安装问题、常见问题FAQ、联系我们
  - 故障排查类：常见故障现象、故障原因分析、排查与解决方案、预防措施、常见问题、联系我们
- [ ] 包含FAQ部分（3-5个问题）
- [ ] 包含CTA引导

### 4. 技术准确性
- [ ] 技术参数与官方PDF核对无误
- [ ] 选型计算公式正确
- [ ] 应用场景描述准确
- [ ] 无错误信息或误导性内容

### 5. 内容专业性
- [ ] 语言风格专业、严谨、实用
- [ ] 适合暖通工程师、采购人员阅读
- [ ] 无错别字、语法错误
- [ ] 标点符号使用正确

## SEO要素审核

### 6. TDK配置
- [ ] 标题包含核心关键词
- [ ] 描述包含核心关键词和品牌
- [ ] 关键词列表完整
- [ ] URL（slug）友好，包含关键词

### 7. 结构化数据
- [ ] Article结构化数据正确
- [ ] FAQPage结构化数据正确（如适用）

### 8. 内链建设
- [ ] 包含相关产品链接
- [ ] 包含相关文章链接
- [ ] 包含联系页面链接

## 品牌与合规审核

### 9. 品牌信息
- [ ] 自然融入品牌身份
- [ ] 公司名称正确：湖北科信达机电设备有限公司
- [ ] 联系电话正确：139-0711-7179
- [ ] 服务范围正确：湖北全省

### 10. 合规性
- [ ] 无虚假宣传
- [ ] 无侵犯版权内容
- [ ] 图片使用合规（有授权或原创）
- [ ] 引用来源标注正确

## 发布前最终检查

### 11. 格式检查
- [ ] Markdown格式正确
- [ ] 图片链接正确
- [ ] 链接可访问
- [ ] 无死链

### 12. 预览检查
- [ ] 在本地预览页面
- [ ] 检查页面显示效果
- [ ] 检查移动端显示
- [ ] 确认无布局问题

## 审核流程

1. **初步审核**：AI生成后，检查基本格式和内容完整性
2. **技术审核**：技术工程师审核技术参数和专业内容
3. **SEO审核**：检查SEO要素和关键词密度
4. **最终审核**：整体检查，确认无误后发布
