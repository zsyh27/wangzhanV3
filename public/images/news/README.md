# 新闻图片库

## 目录结构

```
images/news/
├── hvac/          # 暖通行业动态新闻图片
└── honeywell/     # 霍尼韦尔产品介绍图片
```

## 使用说明

1. **hvac/** - 存放暖通行业相关图片
   - 行业趋势图
   - 市场分析图表
   - 暖通系统示意图
   - 等等

2. **honeywell/** - 存放霍尼韦尔产品相关图片
   - 产品外观图
   - 产品安装图
   - 技术参数图
   - 等等

## 图片要求

- 格式：支持 .jpg, .jpeg, .png, .webp
- 尺寸：建议 800x450（16:9 比例）
- 大小：单张不超过 10MB
- 文件名：使用有意义的英文或拼音命名

## 注意事项

- 确保图片有合法使用权
- 建议为图片添加版权水印
- 图片会自动上传到微信服务号
